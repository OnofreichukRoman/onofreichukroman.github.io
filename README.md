# OnofreichukRoman GitHub Pages

This repository hosts the official Google Play developer website for Roman Onofreichuk.

## Contents

- `index.html` — main developer landing page
- `privacy.html` — dedicated privacy policy page
- `styles.css` — responsive site styling
- `APP_DESCRIPTION.md` — description of the featured app, Meme Soundboard

## About

The website is designed for Google Play developer requirements and includes:

- developer contact information
- official app summary and features
- privacy policy for the app and site
- future app readiness section

## Deployment

This site is hosted with GitHub Pages. To update the site:

1. Edit files in the repository.
2. Commit your changes.
3. Push to the `main` branch.

The site will be available at:

https://onofreichukroman.github.io
